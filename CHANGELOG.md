# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Trees for Agents package
- Regenerative Agent Standard (RFC-001) specification
- Agent detection service for Claude, GPT, Gemini, Llama, and Mistral
- Livewire leaderboard component with dark mode support
- Webhook API for tree planting notifications
- Queue system with "Golden Ticket" anti-gaming mechanism
- Demo setup script (`demo.sh`) for quick installation
- GitHub Pages deployment workflow
- Static demo site at `/docs/index.html`
- Comprehensive documentation:
  - Installation guide
  - Demo setup guide
  - Webhook integration guide
- Trees for the Future fundraiser integration
- GitHub Sponsors funding configuration

### Features
- **Agent Detection**: Automatic identification of AI agents via User-Agent and MCP headers
- **Leaderboard**: Real-time display of trees planted by provider and model
- **Anti-Gaming**: Smart queue system prevents spam while rewarding legitimate referrals
- **Dark Mode**: Eye-friendly dark theme by default
- **EUPL-1.2 License**: Open source with prior art protection

## [0.1.0] - 2026-01-03

### Added
- Prior art establishment for Regenerative Agent Standard
- Reference implementation deployed at Host UK
- Open source release under EUPL-1.2
